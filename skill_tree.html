The GNUBi Skill Tree

A structured path to mastering technology through collaborative learning.

Your Path to Mastery

Phase 1 Foundational Skills

The essential building blocks for any technologist.

Linux CLI Essentials

  • Navigating the filesystem (`ls`, `cd`, `pwd`)
  • File manipulation (`cp`, `mv`, `rm`, `mkdir`)
  • Text processing (`cat`, `grep`, `awk`, `sed`)
  • Permissions (`chmod`, `chown`)

Git & Version Control

  • Basic commands (`init`, `add`, `commit`, `status`)
  • Branching & Merging
  • Remote repositories (`push`, `pull`, `clone`)
  • Resolving conflicts

Networking Basics

  • IP addressing, Subnetting
  • DNS, HTTP/HTTPS
  • Common network tools (`ping`, `traceroute`, `netstat`)

Phase 2 Core Development & Operations

Bridging the gap between development and operations with key tools and practices.

Docker & Containerization

  • Dockerfile creation
  • Image building & management
  • Container orchestration (basic concepts)
  • Docker Compose

CI/CD Pipelines

  • Understanding CI/CD principles
  • Automated testing & deployment
  • Pipeline tools (e.g., Jenkins, GitLab CI, GitHub Actions)

Python Scripting

  • Basic syntax & data types
  • Functions & modules
  • Scripting for automation
  • Interacting with APIs

Phase 3 Cloud & Advanced Topics

Specializing in cloud platforms and advanced architectural patterns.

Kubernetes Fundamentals

  • Pods, Deployments, Services
  • kubectl commands
  • Basic cluster management

Cloud Platform (AWS/Azure/GCP)

  • Core services (compute, storage, networking)
  • Infrastructure as Code (Terraform, CloudFormation)
  • Managed databases

Observability & Monitoring

  • Logging (ELK Stack, Loki)
  • Metrics (Prometheus, Grafana)
  • Tracing (Jaeger, Zipkin)
